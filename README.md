# Controlled-path-Mobile-sink-data-gathering

Wang et al. (2008) presented a model in which two mobile sinks are employed in a grid-based wireless sensor network to enhance the lifetime in an event-driven scenario. Although the sensor field is a square-shaped model and the algorithm imitates a pre-defined movement pattern based on the periphery strategy, the trajectory of the sinks is a diamond-shaped path across the network. Moreover, the sinks move and stop periodically on the opposite sides of each other to cover whole network. Each sensor node tracks the sinks' position and sends its data packets to the nearest sink through multi-hop manner when it detects an event. The authors believe that the proposed strategy successfully distributes the load on the sensor field, hence decreases the delivery latency and improves the network lifetime. (Our implementation may a bit differ from the original work in details)


Wang, B., Xie, D., Chen, C., Ma, J. and Cheng, S. (2008). Deploying Multiple Mobile Sinks in Event-Driven WSNs. Proceedings of the IEEE International Conference on Communications, May 19-23. Beijing: IEEE, 2293-2297.
